# ⌚ Nova Watch

A premium AI-powered smartwatch website built using **HTML, CSS, and Vanilla JavaScript**. Nova Watch showcases a modern product landing experience with elegant UI, responsive layouts, and a complete product purchase flow inspired by premium consumer technology brands.

## 📖 About the Project

Nova Watch is a concept website for a premium AI-powered wellness smartwatch.

The project demonstrates a complete product website experience, including:

- Modern landing page
- Product features showcase
- Premium product purchase page
- Customer reviews
- FAQ section
- Warranty & Support
- Order confirmation experience
- Fully responsive design

The primary goal of this project was to create a premium UI inspired by companies such as Apple, Samsung, and Nothing while using only core web technologies.
